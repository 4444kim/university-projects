# Time Task Manager — функционал и запуск

## Стек

- Next.js 15, React 19, TypeScript
- Zustand (persist в localStorage)
- Tailwind UI kit (shadcn/ui)
- Recharts, react-day-picker

## Как запустить

```bash
pnpm install
pnpm dev
# открыть http://localhost:3000
```

## Данные и хранение

- Всё хранится локально в `localStorage` (ключ `task-manager-storage`).
- Схема мигрирует автоматически (schemaVersion=3).

## Основные разделы

- `/` — менеджер задач (создание, таймер, пауза, завершение, аналитика).
- `/profile` — профиль (ФИО, email) и обзор привычек с календарём.
- `/habits` — управление личными целями: создание, просмотр, календарь, редактирование, удаление.

## Задачи

- Быстрое добавление: название, теги, время (мин), сложность (1–5).
- Карточка задачи: старт/пауза/завершить, таймер, сложности/теги, удалить (кроме задач-привычек).
- Завершённые: история, поиск, редактирование (кроме задач-привычек), удаление.
- Аналитика: время, количество, очки, фокус, топ-теги, экспорт CSV, переключатель звука.
- Звук окончания задачи — встроенный WebAudio (можно отключить).
- Клавиши: `N` — новая задача (фокус на ввод), `S` — старт/пауза, `F` — завершить активную, `Cmd/Ctrl+K` — командная палитра.

## Личные цели (привычки)

- Создание (мастер): введите название, выберите категорию (язык/карьера/учёба/фитнес/другое), частоту (ежедневно, 2/3/4 раза в неделю), время, длительность (1–180 дней). Расписание автогенерируется и видно в превью-календаре.
- На `/habits`: карточки целей с календарём прогресса и подсчётом завершённых дней.
- Редактирование цели: можно менять название, категорию, частоту, время, длительность. Опционально можно включить ручной выбор дней в календаре — расписание и связанные задачи перестраиваются.
- Удаление цели: очищает цель и все связанные задачи (#habit, read-only).
- Интеграция с задачами: каждая привычка создаёт серию задач с тегом `habit` (read-only). Завершение задачи двигает прогресс цели.

## Календарь

- Отображает даты, сгенерированные для привычки (авто или вручную отмеченные).
- Зеленая подсветка — выполненные дни.

## Уведомления

- Системные browser notifications + дублирующий toast.
- Срабатывают:
  - При завершении задачи — “Задача завершена”.
  - По привычкам в заданное время: “Время для привычки: {название} • {время}”.
- Permission запрашивается мягко при первом использовании. Если запрещено — остаются toast-уведомления.

## Профиль

- Поля: ФИО, email. Сохраняется локально.
- Блок «Личные цели» — быстрый просмотр календарей целей.

## Особенности

- Все операции офлайн, без бэкенда.
- Read-only задачи привычек нельзя редактировать/удалять вручную.
- Экспорт CSV для завершённых задач по выбранному периоду (день/неделя/месяц).
- Длительная неактивность: через 10 минут без активных задач показывается ремайндер + опциональное уведомление.

## Тестовый сценарий

1. Открыть `/profile`, заполнить ФИО и email.
2. Перейти `/habits`, создать новую цель (указать время в ближайшую минуту для проверки уведомлений).
3. Убедиться, что в календаре цели появились дни; в списке задач — задачи с тегом `habit`.
4. Запустить и завершить задачу — увидеть уведомление и звук.
5. Дождаться выбранного времени — получить уведомление “Время для привычки” (или хотя бы toast, если permission отклонён).
